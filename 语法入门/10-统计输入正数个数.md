# 统计输入正数个数

## 描述

控制台输入整数，请设计一个死循环，当用户输入非正数时停止输入。请给出用户输入的正整数个数（默认输入个数不超过2147483647）

### 输入描述：

若干整数，整数之间用空格隔开

### 输出描述：

控制台输入的正整数个数

## 示例1

输入：

```
1 2 3 4 5 6 7 0
```

输出：

```
7
```

## 解题

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        int count = 0;
        Scanner scanner = new Scanner(System.in);

        while (true) {
            int num = scanner.nextInt();
            if (num <= 0) {
                break; // 输入非正数时退出循环
            }
            count++;
        }

        System.out.println(count);
    }
}
```
